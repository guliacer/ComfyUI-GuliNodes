# ComfyUI-GuliNodes
面向中文用户习惯的 ComfyUI 实用节点合集，聚焦解决日常出图中重复操作的痛点，覆盖画布排版、节点管理、图像处理、文本交互、模型加载等核心场景。

## 核心功能（优先体验）
### 1. GG Toolbar（画布效率神器）
⭐ 推荐星级：★★★★☆
#### 核心能力
- **节点视觉整理**：支持节点/分组上色（可选择「节点整体/节点内部/标题栏」模式），含预设颜色+2组自定义颜色，支持一键清除颜色；
- **节点批量排版**：等宽/等高、左/右/顶/底对齐、水平/垂直居中、自动间距整理、紧凑化宽度；
- **工具栏自定义**：可拖动位置、右键隐藏/恢复、中键调整背景色/透明度（本地记忆设置）。

#### 快速使用
1. 安装后工具栏默认在画布底部中央，按住空白处 <span style="color:#e63946; font-weight:bold; background:#fef2f2; padding:2px 6px; border-radius:3px;">【鼠标左键】</span> 可拖动；
2. 上色：选颜色 → 点上色按钮 → 点击节点/分组（多选节点时点击任意一个即可批量上色）；
3. 排版：选中多个节点 → 点击对应排版按钮（自动间距需至少3个节点）；
4. 样式设置：<span style="color:#f77f00; font-weight:bold; background:#fff1e6; padding:2px 6px; border-radius:3px;">【鼠标中键】</span> 点击工具栏打开设置面板，支持恢复默认样式；
5. 隐藏工具栏：<span style="color:#457b9d; font-weight:bold; background:#e8f4f8; padding:2px 6px; border-radius:3px;">【鼠标右键】</span> 点击工具栏，<span style="color:#e63946; font-weight:bold; background:#fef2f2; padding:2px 6px; border-radius:3px;">【鼠标左键】</span> 点击迷你图标恢复显示。

### 2. 文本快速接入（GG CLIP文本编码器 + GG 剪贴板读取）
⭐ 推荐星级：★★★★☆
#### 核心能力
- 一键读取系统剪贴板文本到ComfyUI；
- 直接将文本编码为CONDITIONING，快速接入采样流程。

#### 快速使用
1. 外部复制文本 → 用「GG 剪贴板读取」读取，或直接在「GG CLIP文本编码器」点「读取剪贴板」；
2. 仅传文本：从「GG 剪贴板读取」输出字符串；
3. 参与生成：接入CLIP模型，输出CONDITIONING到采样链路。

### 3. 图像压缩+保存（出图交付必备）
⭐ 推荐星级：★★★★☆
#### 核心能力
- 「GG 图像压缩」：支持civilblur（轻量快速）、Caesium（体积精控）、meowtec（画质优先）三种压缩方式，可调质量/目标大小；
- 「GG 图像保存」：支持JPEG/PNG/WEBP/自动格式，承接压缩后的图像保存。

#### 快速使用
图像 → GG 图像压缩（选压缩方式+参数） → GG 图像保存（选格式）。

### 4. GG 网页AI图像反推（网页版AI集成）
⭐ 推荐星级：★★★☆☆
#### 核心能力
- 节点内嵌入豆包、腾讯元宝、文心一言或自定义网页，固定在画布中使用；
- 可调整节点高度适配网页显示，无需配置API Key，复用网页登录态。

#### 注意事项
- 仅作为网页容器，不自动读取第三方网页内容；部分平台因跨域限制无法嵌入，需切换可嵌入网址或单独打开浏览器。

## 安装方式
### 方式1：手动安装
1. 将仓库放入：`ComfyUI/custom_nodes/ComfyUI-GuliNodes`；
2. 安装依赖：`pip install -r requirements.txt`；
3. 重启ComfyUI。

### 方式2：Git安装
```bash
cd ComfyUI/custom_nodes
git clone https://github.com/guliacer/ComfyUI-GuliNodes.git
cd ComfyUI-GuliNodes
pip install -r requirements.txt
```

## 依赖说明
核心依赖：torch、numpy、pillow、opencv-python、llama-cpp-python、mediapipe；
- 无需额外安装pyperclip（剪贴板功能前端实现）；
- aiohttp/comfy等由ComfyUI环境提供，无需单独安装；
- 不锁定依赖版本，兼容主流Python/ComfyUI环境。

## 注意事项
- 视频节点413报错：因文件上传过大，`413` 表示 `Content Too Large`，也就是浏览器上传到 ComfyUI 的单次请求体超过了服务端允许的大小，建议将视频移到ComfyUI/input目录加载；
- 网页AI反推节点：仅嵌入网页，不读取第三方平台内容，部分平台因跨域无法嵌入；
- 如有节点显示/交互问题，可提交Issue反馈。
