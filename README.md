# ComfyUI-GuliNodes

专注于尺寸适配、图像对比和工具类的 ComfyUI 插件。

## 功能列表

### 1. 尺寸工具
- GG 比例适配计算器 - 计算符合8倍数的宽高
- GG 比例适配 Latent 生成器 - 生成对应尺寸空 Latent
- GG 比例适配 Latent (可接图像) - 根据输入图像生成 Latent

### 2. 图像工具
- GG RGBA转RGB - 移除Alpha通道

### 3. 对比工具
- GG 图像对比 4张 - 横向拼接4张图像
- GG 图像对比 8张 - 横向拼接8张图像

### 4. LoRA工具
- GG LoRA 文件选择堆 4个 - 加载4个LoRA
- GG LoRA 文件选择堆 8个 - 加载8个LoRA

### 5. 编组控制
- GG 多组控制 - 批量控制所有编组
- GG 单组控制 - 控制单个编组

### 6. 工具
- GG 文本合并 - 合并文本
- GG 随机种 - 随机种子生成

## 安装

1. 放入 `ComfyUI/custom_nodes/`
2. 重启 ComfyUI
3. 输入 `GG` 搜索节点
