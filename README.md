# ComfyUI-GuliNodes

本插件部分功能参考/来源于各位开源大佬的插件，仅做了整合和适配工作。如有侵权请联系删除。

## 功能列表

### 1. 工具栏（Toolbar）
工作流编辑辅助工具栏，浮动在界面底部。

#### 功能按钮
| 按钮 | 功能 | 说明 |
|------|------|------|
| 自动宽度 | ⬜ | 将选中节点统一为相同宽度 |
| 自动高度 | ▬ | 将选中节点统一为相同高度 |
| 最左对齐 | ⬅ | 节点左边缘对齐 |
| 最右对齐 | ➡ | 节点右边缘对齐 |
| 水平居中 | ⬌ | 节点垂直居中对齐 |
| 最顶对齐 | ⬆ | 节点顶部对齐 |
| 最底对齐 | ⬇ | 节点底部对齐 |
| 垂直居中 | ⬆⬇ | 节点水平居中对齐 |
| 自动间距 | ↔ | 等间距排列节点 |
| 自适应尺寸 | ▢ | 压缩节点为紧凑尺寸 |

#### 使用方法
1. **选中节点**：按住Ctrl点击或框选多个节点
2. **点击按钮**：对选中节点执行对应操作
3. **拖动工具栏**：左键按住工具栏主体拖动
4. **隐藏/显示**：右键点击工具栏或小图标

### 2. 尺寸工具
- GG 比例适配计算器 - 输入宽高比和边长，计算符合8倍数的输出尺寸
- GG 比例适配 Latent 生成器 - 生成指定尺寸的空Latent用于提示生成
- GG 比例适配 Latent (可接图像) - 根据输入图像自动匹配尺寸生成Latent

### 3. 图像工具
- GG RGBA转RGB - 移除Alpha透明通道，输出纯RGB图像

### 4. 对比工具
- GG 图像对比 4张 - 横向拼接4张图像并显示标签
- GG 图像对比 8张 - 横向拼接8张图像并显示标签

### 5. LoRA工具
- GG LoRA 文件选择堆 4个 - 依次应用多个LoRA，支持选择"无"跳过
- GG LoRA 文件选择堆 8个 - 依次应用多个LoRA，支持选择"无"跳过

### 6. 编组控制
- GG 多组控制 - 批量启用/跳过工作流中的所有编组
- GG 单组控制 - 控制单个编组的启用/跳过状态

### 7. 工具
- GG 文本合并 - 合并多段文本，可自定义分隔符
- GG 随机种 - 生成随机种子，支持多种模式

## 安装

1. 放入 `ComfyUI/custom_nodes/`
2. 重启 ComfyUI
3. 输入 `GG` 搜索节点
