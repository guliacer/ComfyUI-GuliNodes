# ComfyUI-GuliNodes

本插件提供了一系列实用的 ComfyUI 自定义节点，旨在提高工作流编辑效率和扩展功能。部分功能参考/来源于开源社区，仅做了整合和适配工作。如有侵权请联系删除。

## 功能分类

### 🛠️ 工具栏（Toolbar）
工作流编辑辅助工具栏，浮动在界面底部，提供节点排列和对齐功能。

#### 功能按钮
| 按钮 | 功能 | 说明 |
|------|------|------|
| 自动宽度 | ⬜ | 将选中节点统一为相同宽度 |
| 自动高度 | ▬ | 将选中节点统一为相同高度 |
| 最左对齐 | ⬅ | 节点左边缘对齐 |
| 最右对齐 | ➡ | 节点右边缘对齐 |
| 水平居中 | ⬌ | 节点垂直居中对齐 |
| 最顶对齐 | ⬆ | 节点顶部对齐 |
| 最底对齐 | ⬇ | 节点底部对齐 |
| 垂直居中 | ⬆⬇ | 节点水平居中对齐 |
| 自动间距 | ↔ | 等间距排列节点 |
| 自适应尺寸 | ▢ | 压缩节点为紧凑尺寸 |

#### 使用方法
1. **选中节点**：按住Ctrl点击或框选多个节点
2. **点击按钮**：对选中节点执行对应操作
3. **拖动工具栏**：左键按住工具栏主体拖动
4. **隐藏/显示**：右键点击工具栏或小图标

### 📏 图像尺寸工具
- **GG 图像比例** - 输入宽高比和边长，计算符合8倍数的输出尺寸
- **GG Latent** - 生成指定尺寸的空Latent用于提示生成
- **GG 图像-Latent** - 根据输入图像自动匹配尺寸生成Latent

### 🖼️ 图像工具
- **GG RGBA转RGB** - 移除Alpha透明通道，输出纯RGB图像

## 详细使用说明

### 剪贴板读取使用说明
**功能**：快速读取系统剪贴板中的文本内容，自动显示在节点中并通过输出端口传递给其他节点。

**使用场景**：
- **文生图提示词优化**：使用豆包、千问、Gork等AI工具修改提示词后，无需手动全选复制，直接点击节点按钮即可快速更新
- **配置信息导入**：快速导入预设配置、参数设置等文本内容
- **批量文本处理**：与其他文本处理节点配合使用，提高工作流效率

**使用方法**：
1. 在其他软件/平台中复制需要的文本内容
2. 在ComfyUI中添加「GG 剪贴板读取」节点
3. 读取的内容会显示在文本框中，并通过输出端口传递给其他节点（如提示词输入节点）
4. 如需编辑内容，可直接在文本框中修改

**优势**：
- 无需手动全选和复制粘贴，减少重复操作
- 支持多行文本，适合处理长提示词
- 与其他节点无缝集成，提高工作流效率

### 图像反推使用说明
**功能**：使用多模态模型对图像进行分析，生成详细的描述和提示词。

**使用场景**：
- **提示词生成**：根据图像自动生成高质量的提示词
- **图像分析**：获取图像的详细描述和特征
- **内容理解**：理解图像中的场景、物体和情感

**使用方法**：
1. 在ComfyUI中添加「GG 模型加载器」节点
2. 选择模型类型（Qwen3-VL、Qwen3.5-VL、Gemma4）
3. 选择主模型文件和视觉投影mmproj文件（多模态模型需要）
4. 添加「GG 图像推理」节点，连接模型输出
5. 选择输入模式（图片或文本）
6. 连接输入图像或文本
7. 调整推理参数（最大边长、最大生成token、温度等）
8. 运行工作流，获取模型生成的文本输出

## 安装

1. **下载插件**：将本插件放入 `ComfyUI/custom_nodes/` 目录
2. **安装依赖**：确保已安装 `llama-cpp-python` 库（用于图像反推功能）
3. **重启 ComfyUI**：使插件生效
4. **查找节点**：在节点搜索框中输入 `GG` 查找所有节点

## 测试

插件包含 `test_import.py` 测试脚本，用于验证插件是否正确导入和注册所有节点。运行方法：

```bash
python test_import.py
```
